Endpoints & Modelos

Documentação técnica para integração com o backend Supabase do PloutosLedger.

CashFlow (Fluxo de Caixa)

GET /rest/v1/ploutos_cashflow

Retorna todas as transações do usuário autenticado.

Parâmetros de Query

  • select: Campos a retornar (padrão: *)
  • type: Filtrar por 'entry' ou 'exit'
POST /rest/v1/ploutos_cashflow

Cria uma nova transação financeira.

Request Body (JSON)

{
  "amount": 150.50,
  "type": "entry",
  "description": "Venda de Produto A",
  "category_id": "vendas_diretas",
  "tags": ["online", "cartao"]
}

Modelos de Dados (Schemas)

Campo Tipo Descrição
id UUID Identificador único universal.
amount Decimal Valor monetário com 2 casas decimais.
user_id UUID Referência ao usuário no Auth do Supabase.
